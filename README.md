## Instruktioner

För att göra ändringar i det här repot så behöver du först bli tillagd som collaborator.

När du är tillagd så kan du enklet publicera en ny version av spelet genom att ladda upp den nya html-filen. Den ska heta index.html.

Du behöver även se till att nya bilder läggs till i mappen bilder så att de går att hitta.

Se till att du commitar dina changes till 'main' och sedan autogenereras sidan på någon minut.
